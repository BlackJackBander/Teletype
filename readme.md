# TeleType Editor для Firefox

![TeleType Editor]

**TeleType Editor** — продвинутый текстовый редактор с подсветкой синтаксиса для браузера Firefox.

## Возможности ✨

- 📝 **Богатое форматирование текста**
- 💻 **Подсветка синтаксиса** для 15+ языков программирования
- 🌙 **Темная тема** для комфортной работы
- 💾 **Автосохранение** документов
- 📤 **Экспорт в HTML**
- ⚡ **Горячие клавиши** для быстрой работы
- 🔧 **Управление документами** (создание, поиск, импорт)

## Установка 🔧

1. Скачайте архив с расширением
2. Откройте Firefox и перейдите на страницу `about:debugging`
3. Нажмите "This Firefox" → "Load Temporary Add-on"
4. Выберите файл `manifest.json` из папки с расширением

## Разработка 🛠️

### Технологии:
- HTML5, CSS3, JavaScript
- Highlight.js для подсветки синтаксиса
- Chrome Extension API (совместимо с Firefox)
